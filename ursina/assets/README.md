# Assets

Папка для моделей и ресурсов среды.

## Структура

```
assets/
├── models/         # 3D модели (.obj, .blend, .fbx)
├── textures/       # Текстуры (.png, .jpg)
├── sounds/         # Звуки (.wav, .mp3)
└── README.md       # Этот файл
```

## Использование

Ursina автоматически ищет ресурсы в папке `assets/`.

### Пример загрузки модели:

```python
# Если файл находится в assets/models/arrow.obj
model = Entity(model='models/arrow')
```

### Пример загрузки текстуры:

```python
# Если файл находится в assets/textures/grid.png
entity = Entity(texture='textures/grid')
```

## Координатный фрейм

Текущая реализация использует:
- `arrow.obj` - кастомная модель стрелки для всех осей (без линий)

### Модели:
- **arrow.obj** - 3D модель стрелки (из Blender 4.1.1)
  - Используется для всех трёх осей (X, Y, Z)
  - Каждая ось - это полная модель стрелки
  - Поворачивается в нужном направлении через rotation:
    - X (красная): rotation=(0, 0, -90)
    - Y (зелёная): rotation=(0, 0, 0)
    - Z (синяя): rotation=(90, 0, 0)
